# Elicitation（引导技巧）

通过自然对话进行心理画像分析。

## 目标

在审查或创建与心理引导相关的内容时——无论是对话脚本、访谈问题、用户研究流程，还是实现这些技术的任何代码——**你的目标是达到 10/10 分**。

根据本技能中的原则和技术，按 0-10 的标准对所有工作进行评分。以 **X/10** 的格式提供你的评估，并具体说明哪些方面做得好，哪些方面需要改进才能达到 10/10。

10/10 分意味着该工作：
- 体现了核心原则（深度来自耐心，而非探查）
- 使用了研究传统中的适当技术
- 避免了所有反模式
- 创造了真实自我披露的条件

不断迭代直到达到 10/10。

## 核心原则

**深度来自耐心，而非探查。**

最具揭示性的信息在人们感到安全分享时出现，而非在被质问时。你的角色是创造对话空间，让自我披露感觉自然而有意义。

关键洞察：人们想要讲述自己的故事。他们很少有机会这样做。

## 三大研究传统

本技能综合了三种互补的方法：

### 1. 自传式记忆研究
记忆如何塑造身份。关键发现：**自我定义记忆**(Singer)是人格的基石——生动、情感强烈、频繁被回忆起、与持久关切相关的记忆。

### 2. 叙事身份理论
人们如何构建生活故事。关键发现：人们使用的**叙事主题**（救赎 vs. 污染，能动性 vs. 共融性）比实际事件本身更能预测心理健康(McAdams)。

### 3. 动机性访谈
如何在不产生阻抗的情况下促进披露。关键发现：**反思优于提问**，能引发更真实的自我披露。目标反思与提问比率为 2:1(Miller & Rollnick)。

## 自我定义记忆

Jefferson Singer 确定了使记忆"自我定义"的五个标准：

1. **生动** - 丰富的感官和情感细节
2. **情感强烈** - 强烈的感受，积极或消极
3. **频繁回顾** - 经常浮现脑海，向他人讲述
4. **与相似记忆关联** - 是某个模式或主题的一部分
5. **与持久关切相连** - 反映持续的目标、冲突或未解决的问题

### 引导自我定义记忆

不要问："你最具塑造性的记忆是什么？"

相反，创造对话框架：

**"不断回来"框架：**
> "有些记忆会一直伴随我们——它们在意想不到的时刻突然浮现，或者我们发现自己会向生活中的新朋友讲述它们。你有这样的记忆吗？"

**"解释我是谁"框架：**
> "当你认识新朋友并希望他们真正理解你的来历时，有没有一个故事或时刻是你会分享的？"

**"转折点"框架：**
> "回顾过去，有没有一个时刻感觉事情发生了转变——生活在那之前和之后感觉有所不同？"

### 自我定义记忆揭示的内容

| 记忆特征 | 人格洞察 |
|----------------|---------------------|
| 掌控、成就主题 | 高能动性需求 |
| 连接、关系主题 | 高共融性需求 |
| 救赎序列（坏→好） | 韧性、生成性 |
| 污染序列（好→坏） | 抑郁风险、未解决的创伤 |
| 整合与意义建构 | 心理成熟 |
| 碎片化与混乱 | 身份扩散 |

参见：[自我定义记忆参考](elicitation/self-defining-memories.md)

## 生活故事访谈：8个关键场景

Dan McAdams 的生活故事访谈要求描述 8 个揭示叙事身份的特定"场景"：

1. **高峰** - 巅峰体验，最美妙的时刻
2. **低谷** - 最低点，最困难的时刻
3. **转折点** - 标志着人生重大变化的时刻
4. **最早的记忆** - 第一个清晰的记忆
5. **重要的童年记忆** - 12岁前的生动记忆
6. **重要的青春期记忆** - 青少年时期的生动记忆
7. **重要的成年记忆** - 最近的重要记忆
8. **其他重要记忆** - 任何其他定义你是谁的记忆

### 对话式改编

你不需要按顺序询问全部 8 个。相反：

**以好奇心开场，而非议程：**
> "我对塑造你的那些时刻很好奇。不一定是那些简历上的大事——更多的是那些一直伴随你的经历。"

**跟随他们的引导：**
当他们提到某个人生阶段时，温和地探索：
> "那段时间对你来说是什么样的？有什么特别突出的时刻吗？"

**跨越时间桥接：**
> "那听起来很重要。有没有更早的——或更晚的——时刻与那种感觉相关联？"

### 要倾听的叙事主题

**能动性主题**（个人力量、成就、掌控）：
- "我决定..."
- "我让它发生..."
- "我坚持下来..."

**共融性主题**（连接、爱、归属）：
- "我们都在一起..."
- "我感到如此亲近..."
- "他们理解我..."

**救赎序列**（苦难带来成长）：
- "那很糟糕，但..."
- "回头看，我很高兴..."
- "那造就了现在的我..."

**污染序列**（好变坏）：
- "事情很好直到..."
- "我以为我很快乐，但..."
- "它毁了一切..."

参见：[叙事身份参考](elicitation/narrative-identity.md)

## OARS 框架

动机性访谈的核心技能，改编用于引导：

### 开放式问题（Open Questions）
不能用是/否回答的问题。但要少用。

不要："你喜欢你的童年吗？"
试试："在你的家庭中长大是什么样的？"

### 肯定（Affirmations）
真诚认可优点、努力或价值观——而非赞美。

不要："太棒了！"
试试："即使代价高昂，你也重视诚实。"

### 反思（Reflections）
重述或重新表述他们所说的话。这是核心技能。

**简单反思**（复述）：
> "所以你在那一刻感到被忽视。"

**复杂反思**（增加意义）：
> "听起来被认可对你真的很重要——你需要知道你的贡献被看见。"

**放大反思**（温和夸张）：
> "所以无论他们做什么都不会有任何改变。"（通常会促使他们细化立场）

**双面反思**（同时持有两种真相）：
> "一方面，你喜欢那种稳定。另一方面，你感到被困住了。"

### 总结（Summaries）
定期汇总你听到的内容。创造意义并邀请修正。

> "让我看看我有没有理解：成长过程中，你学会了自力更生，因为寻求帮助意味着失望。但你也注意到这种模式现在让人们与你保持距离。你在想是否有另一种方式。"

### 2:1 比率

**目标是每个问题配两个反思。**

问题收集信息但可能感觉像审问。反思显示理解并邀请详述。

不好的模式：
> 问："发生了什么？" → 问："你感觉如何？" → 问:"你接下来做了什么？"

更好的模式：
> 问："发生了什么？" → 反思："那让你措手不及" → 反思："你不确定怎么理解它"

参见：[动机性访谈参考](elicitation/motivational-interviewing.md)

## 价值观引导

Shalom Schwartz 的 10 项普世价值观为理解动机提供了框架：

| 价值观 | 核心关切 |
|-------|--------------|
| **自我导向** | 独立性、自由、创造力 |
| **刺激** | 新奇、兴奋、挑战 |
| **享乐主义** | 快乐、享受、满足 |
| **成就** | 成功、能力、抱负 |
| **权力** | 权威、财富、社会地位 |
| **安全** | 安全、稳定、秩序 |
| **遵从** | 服从、自律、礼貌 |
| **传统** | 尊重、承诺、谦逊 |
| **仁慈** | 乐于助人、忠诚、宽恕 |
| **普世关怀** | 平等、正义、环境保护 |

### 价值观引导技巧

**榜样技巧：**
> "你钦佩谁？具体来说，你欣赏他们什么？"

**反向日技巧：**
> "你永远不可能成为什么样的人？什么会让你感觉像是对自己的背叛？"

**决策考古：**
> "想想你做过的一个艰难选择。最终是什么让天平倾斜的？"

**愤怒作为价值观信号：**
> "什么会让你真正愤怒——不是烦恼，而是道德上的愤慨？"

参见：[价值观引导参考](elicitation/values-elicitation.md)

## 图式检测

Jeffrey Young 的 18 项早期适应不良图式是从童年发展并在不同情境中持续存在的稳定思维和感受模式：

### 五大领域

**1. 断裂与拒绝**
- 遗弃、不信任/虐待、情感剥夺、缺陷/羞耻、社会隔离

**2. 受损的自主性**
- 依赖/无能、易受伤害、纠缠/未发展的自我、失败

**3. 受损的界限**
- 特权/自大、自控不足

**4. 他人导向**
- 屈从、自我牺牲、寻求认可

**5. 过度警觉与抑制**
- 消极/悲观、情感抑制、苛刻标准、惩罚性

### 向下箭头技巧

当某人表达表面担忧时，温和探查更深层的信念：

> 对方："我担心这次演讲。"
> 你："最坏会发生什么？"
> 对方："我可能在所有人面前搞砸。"
> 你："如果那发生了，会意味着什么？"
> 对方："他们会看到我不知道自己在做什么。"
> 你："那对你来说意味着什么？"
> 对方："我是个骗子。我不配在这里。"

箭头底部通常揭示一个图式（这里是：缺陷/羞耻或失败）。

### 图式的语言标记

| 图式 | 语言模式 |
|--------|-------------------|
| 遗弃 | "最终每个人都会离开..." |
| 缺陷 | "我有问题..." |
| 失败 | "我从不完成任何事..." |
| 情感剥夺 | "没人真正理解..." |
| 苛刻标准 | "永远不够好..." |

参见：[图式检测参考](elicitation/schema-detection.md)

## 回忆高峰

人们在 10-30 岁期间拥有更多且更生动的记忆（"回忆高峰"）。这是身份形成的时期。

**瞄准高峰：**
- 初恋
- 第一份工作或职业定义时刻
- 离家
- 形成的关键友谊
- 教育转折点
- 早期成年的挣扎与胜利

**从现在桥接到高峰：**
> "你提到在工作中感觉像个局外人。在生命早期——也许在学校或刚开始工作时——有没有感受过类似的事情？"

## 按人生阶段划分的问题序列

Barbara Haight 的生活回顾访谈提供了结构化序列：

### 童年（12岁前）
1. 你的家是什么样的？
2. 你的父母是什么样的？
3. 你在家庭中的角色是什么？
4. 你小时候是什么样的？
5. 你最喜欢做什么？

### 青春期（12-18岁）
1. 你的身体如何变化？你对此感觉如何？
2. 学校对你来说是什么样的？
3. 你的友谊是什么样的？
4. 你梦想成为什么？
5. 作为青少年最困难的是什么？

### 早期成年（18-30岁）
1. 离家是什么样的？
2. 你最初的认真关系是什么样的？
3. 你做什么工作，感觉如何？
4. 那段时间你的目标是什么？
5. 你面临的最大挑战是什么？

### 中年（30-60岁）
1. 你对自己的认知如何变化？
2. 你的主要成就是什么？
3. 你经历了哪些失去？
4. 你的关系如何演变？
5. 你对自己有什么新的了解？

### 晚年（60岁以上）
1. 你的日常生活如何改变？
2. 现在对你来说什么最重要？
3. 你想留下什么遗产？
4. 你现在理解了什么以前不理解的？
5. 你会对年轻时的自己说什么？

参见：[问题序列参考](elicitation/question-sequences.md)

## 按主题划分的敏化问题

James Birren 的引导式自传使用主题提示：

### 家庭主题
- 你家的情感氛围是什么样的？
- 你和谁最亲近？和谁冲突？
- 家里反复讲述什么故事？

### 工作主题
- 除了赚钱，工作对你意味着什么？
- 你何时在职业上感到最充实？
- 即使不付钱你也会做什么工作？

### 金钱主题
- 你家庭中关于金钱的信息是什么？
- 经济安全对你意味着什么？
- 如果钱不是问题，你会做什么？

### 健康主题
- 你与身体的关系如何变化？
- 什么健康经历影响了你对生活的看法？
- 你如何照顾自己？

### 死亡主题
- 你经历过重大损失吗？
- 关于死亡的想法如何影响你的生活方式？
- 你想被记住什么？

### 意义主题
- 什么给你的生活以意义？
- 什么信念或价值观指导你？
- 你还在试图回答什么问题？

## 人格的语言标记

LIWC（语言探究与字数统计）研究识别模式，**但要谨慎使用**：

| 模式 | 可能的指示 |
|---------|---------------------|
| 高"我"使用 | 自我关注、可能抑郁、诚实 |
| 高"我们"使用 | 集体主义倾向、亲密 |
| 负面情感词 | 痛苦，但也在处理 |
| 认知复杂性词（因为、想、知道） | 分析思维、意义建构 |
| 现在时态聚焦 | 即时性、可能冲动 |
| 过去时态聚焦 | 反思、可能反刍 |

### 关键警告

1. **情境极其重要。**相同的词语模式在不同情境中意义不同。
2. **交叉验证。**永远不要仅依赖语言。用行为和明确陈述进行三角验证。
3. **总体而非个体。**LIWC 发现是关于群体平均值的。个体差异巨大。
4. **文化差异。**词语使用规范在不同文化和语言中差异显著。

参见：[语言推断参考](elicitation/language-inference.md)

## 反模式

**不该做的事：**

### 审问陷阱
连珠炮式提问感觉像面试，而非对话。人们会变得戒备。

相反：放慢节奏。多反思，少提问。

### 解释跳跃
在有证据前就跳到心理结论。

相反：轻松持有假设。寻找反证。

### 议程推动
引导向你认为重要的话题，而非跟随他们的能量。

相反：让他们引导。他们的重点就是数据。

### 过早深入
在建立信任前询问深层私人问题。

相反：逐步赢得披露。从容易的领域开始。

### 治疗角色扮演
使用临床语言或暗示你在治疗他们的技术。

相反：保持好奇，而非临床。你是在了解他们，而非诊断。

### 独白回应
用自己的冗长故事回应他们的披露。

相反：保持焦点在他们身上。简短的自我披露可以建立融洽关系，但始终回到他们。

### 验证陷阱
为了维持融洽关系而同意一切。

相反：真诚的反思可以温和地挑战而不对抗。
