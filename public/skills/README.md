# Skills 目录

本目录包含可公开分享的通用 Skills。

## 目录结构

```
skills/
├── README.md           # 本文件
└── [skill-name]/       # 各个 skill 目录
    ├── SKILL.md        # Skill 描述
    ├── metadata.json   # 元数据
    └── resources/      # 资源文件
```

## 如何添加 Skill

1. 在此目录下创建新的 skill 文件夹
2. 添加 SKILL.md 和 metadata.json
3. 更新本 README.md

## 注意事项

- 只包含可公开分享的通用 skills
- 个人定制的 skills 应放在 `private/custom-skills/`
- 确保不包含敏感信息
