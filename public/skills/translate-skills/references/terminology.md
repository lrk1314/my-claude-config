# 技术术语翻译参考

本文档提供 Claude 技能和开发相关的常见技术术语标准中文翻译。

## 技能相关术语

| 英文 | 中文 | 说明 |
|------|------|------|
| Skill | 技能 | Claude 的模块化功能扩展 |
| Frontmatter | 前置元数据 / Frontmatter | YAML 格式的元数据区域 |
| Bundled resources | 捆绑资源 | 技能附带的文件资源 |
| Progressive disclosure | 渐进式展示 | 分层加载信息的设计原则 |
| Workflow | 工作流程 | 多步骤的操作流程 |
| Tool integration | 工具集成 | 与外部工具的集成 |
| Domain expertise | 领域专业知识 | 特定领域的专业知识 |

## 翻译注意事项

1. **保持一致性**: 在同一文档中使用相同术语的统一翻译
2. **技术准确性**: 某些术语如 API、YAML、Markdown 通常保持英文
3. **语境适配**: 根据上下文选择最合适的翻译
4. **专业性**: 使用技术社区广泛接受的译法
