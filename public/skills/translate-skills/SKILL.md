# 将技能翻译为中文

将技能文档从英文翻译为中文,保持原始结构和技术规范。

## 核心原则

### 1. 保留结构
- 保持目录结构和文件名不变
- 不重命名 SKILL.md 文件
- 保留 YAML frontmatter 的所有元数据字段
- 保留代码块、链接和格式

### 2. 翻译质量
- 用准确、自然的中文翻译描述性和说明性文本
- 更新 frontmatter 中的 `description` 字段
- 保持技术术语的一致性
- 翻译所有标题和列表项

### 3. 术语一致性
参考 [terminology.md](references/terminology.md) 获取常用技术术语的标准翻译。在整个文档中保持术语的一致使用。

### 4. 元数据处理
- **name** 字段：通常保持英文(除非使用中文命名约定)
- **description** 字段：翻译为中文,确保清晰完整
- 其他字段：保持不变

## 翻译工作流程

1. 读取英文 SKILL.md 文件
2. 翻译 description 字段和所有内容
3. 参考术语表保持技术术语一致
4. 验证 Markdown 格式和链接完整性
5. 保存翻译后的文件
